# School Management System – Python & MySQL Project

##  Project Overview
This project is a **console-based School Management System** developed using **Python and MySQL**.  
It manages school operations such as **student admissions, fees management, teacher records, and class schedules** using a relational database.

The project demonstrates **Python–SQL integration** with multiple user roles.


## Project Objectives
- Manage student admissions and records
- Handle student fee payments and history
- Maintain teacher information
- Create and view class schedules
- Implement role-based access (Admin, Student, Teacher, Accountant)


##  Technologies Used
- **Python**
- **MySQL**
- **MySQL Connector for Python**
- **SQL (CRUD Operations)**

##  Project Workflow
1. User selects role (Admin / Student / Teacher / Accountant)
2. User logs in using role-based password
3. Menu-driven operations are displayed
4. Python executes SQL queries on MySQL database
5. Data is stored and retrieved in real-time
